
# TFTP

```
client usage: ./gradlew run --client  
 -h,--host <arg>                        server hostname
 -p,--port <arg>                        server port
 -o,--operation <GET|PUT> <src> <dst>   operation
 -m,--mode <NETASCII|OCTET>             file mode

server usage: ./gradlew run --server
 -p,--port <arg>   server port
```
